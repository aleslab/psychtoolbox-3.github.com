# [MeasXYZ](MeasXYZ)
##### >[Psychtoolbox](Psychtoolbox)>[PsychHardware](PsychHardware)

 [XYZ,qual] = [MeasXYZ](MeasXYZ)([meterType])  
  
 Measures XYZ tristimulus coordinates, luminance in  
 cd/m^2.  
  
        meterType == 1:  PR650 (default).  
       meterType == 3:  CRS colorimeter.  
        meterType == 4:  PR655.

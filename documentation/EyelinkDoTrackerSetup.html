USAGE: result=EyelinkDoTrackerSetup(el [, sendkey])

    el: Eyelink default values  
    sendkey: set to go directly into a particular mode  
            'v', start validation  
            'c', start calibration  
            'd', start driftcorrection  
            13, or el.ENTER\_KEY, show 'eye' setup image

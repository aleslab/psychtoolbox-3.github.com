Datapixx(‘SetDinDataDirection’)

[[Psychtoolbox]] › [[Datapixx]].{mex*,dll} subfunction

Although DIN is primarily intended to be a digital _input_ system, it is
possible to turn around these I/Os, to drive outputs at up to 3.3V. To configure
a bit as an output, set the corresponding bit in “directionMask”.
See DatapixxSimonGame, DatapixxDin*Demo files for examples.

Datapixx(‘StopDacSchedule’)

[[Psychtoolbox]] › [[Datapixx]].{mex*,dll} subfunction

Stop running a DAC waveform playback schedule. The actual hardware schedule will
be terminated by the next RegWr* command.
Note that if maxScheduleFrames was assigned a non-0 value in SetDacSchedule,
then the schedule can be left to terminate automatically without having to call
StopDacSchedule.
See DatapixxDac*Demo files for examples.

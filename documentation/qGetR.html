qGetR: get a 3x3 rotation matrix R from a Quaternion Qrotation.

R = qGetR( Qrotation )

IN:

Qrotation - quaternion (x,y,z,w) describing rotation

OUT:

R - 3 x 3 rotation matrix

VERSION: 03.03.2012

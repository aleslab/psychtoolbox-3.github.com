PsychOculusVRCore(‘Close’)

[[Psychtoolbox]] › [[PsychOculusVRCore]].{mex*,dll} subfunction

Close connection to Oculus Rift device ‘oculusPtr’. Do nothing if no such device
is open.
If the optional ‘oculusPtr’ is omitted, then close all open devices and shutdown
the driver, ie. perform the same cleanup as if ‘clear PsychOculusVRCore’ would
be executed.
